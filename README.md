# Sleepable
A parody corporate-dystopian portal for a mandatory dream monetization service. The world's first sleep control platform.

## ✨ Features 
- **🌍 Bilingual Support:** Seamlessly toggle between English and Turkish via URL parameters.
- **⚡ Jumpscare Glitch:** A rare, violent text-glitch animation that cycles every 90 seconds.
- **🌓 Persistence:** Remembers your language choice across all 3 pages using local logic.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for pricing, contact, and landing pages.
- **CSS3:** Custom variables, glassmorphism, and complex keyframe animations.
- **Vanilla JavaScript:** Lightweight state management for localization.
